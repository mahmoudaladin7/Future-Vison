# FUTURE Vision — Consulting & Investment Website (Next.js)

A premium, scalable foundation that upgrades a landing page into a real website with:
- Home, Consulting, Investment, Portfolio, Insights, Contact, Deck Request
- About, Careers, Privacy, Terms
- Modern UI (Tailwind), motion (Framer Motion), icons (Lucide)
- Forms ready for wiring to your email/CRM (React Hook Form + Zod)

## Quick start

```bash
npm install
npm run dev
```

Open: http://localhost:3000
